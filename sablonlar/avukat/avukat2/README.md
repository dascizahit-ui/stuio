# Lexora Hukuk Web Sitesi

Modern avukatlık bürosu sitesi. HTML, CSS ve JavaScript ile hazırlanmıştır.

## Dosyalar
- `index.html`: Ana sayfa
- `uzmanlik.html`: Uzmanlık alanı detay sayfası. Örnek: `uzmanlik.html?alan=sirketler`
- `makale.html`: Makale detay sayfası. Örnek: `makale.html?slug=kvkk-veri-ihlali`
- `styles.css`: Responsive tasarım ve tüm stiller
- `app.js`: Mobil menü, makale arama, iletişim formu, detay sayfası içerikleri
- `assets/`: Logo, hero görseli ve ekip avatarları

## Notlar
Ekip avatarları yerel SVG olarak hazırlanmıştır; dış görsel bağlantısı yoktur. Böylece dosyalar taşındığında görseller bozulmaz.
